# Finding Hummingbot API Service Name

**Current:** Seeing Postgres networking info  
**Need:** Hummingbot API service name

---

## üîç **How to Find Service Name**

### **Option 1: Check Service List**

1. **Go back to Railway project overview**
   - Click on your project name (top left)
   - You should see list of services:
     - Trading Bridge
     - Hummingbot API (or whatever you named it)
     - Postgres

2. **Check the service name:**
   - What is the Hummingbot API service called?
   - Common names: `hummingbot-api`, `hummingbot`, or auto-generated

---

### **Option 2: Check Variables Tab**

1. **In Hummingbot API service**
2. **Go to Variables tab**
3. **Look for:**
   - `RAILWAY_SERVICE_NAME` - This is the service name!
   - OR check other Railway variables

---

### **Option 3: Check Settings**

1. **In Hummingbot API service**
2. **Settings tab** (not Networking)
3. **Look for:**
   - Service name
   - Service ID
   - Any identifier

---
